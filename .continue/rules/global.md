## Coding Standards

- Use modern Python type hints and FastAPI dependency injection.
- Validate request payloads and format responses using Pydantic schemas.
- Use asynchronous operations (async/await) for database queries and non-blocking I/O.
- Maintain a strict separation of concerns across Models, Schemas, and Endpoints.
- Follow the existing nested router patterns when adding new domain modules.
- Use .env file for environment variables.
- Extend CommonModel for new models.
